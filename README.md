🏥 Healthcare Analytics Dashboard

An interactive Healthcare Analytics Dashboard built using Microsoft Power BI to analyze patient health records and generate meaningful healthcare insights through interactive visualizations.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 Project Overview

This project focuses on analyzing healthcare data to identify trends related to patient health, overweight status, heart attacks, strokes, and health risks. The dashboard enables users to explore data through interactive visuals and filters, supporting better data-driven decision-making.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🎯 Objectives

• Analyze patient health records.
• Monitor overweight patients by gender.
• Understand health risk distribution.
• Analyze heart attack and stroke cases.
• Study the relationship between exercise habits and health risks.
• Build an interactive healthcare dashboard using Power BI.

📊 Dashboard Features

• Total Patient Count KPI
• Female Overweight Analysis
• Female Not Overweight Analysis
• Male Overweight Analysis
• Male Not Overweight Analysis
• Stroke Distribution
• Health Risk Distribution
• Heart Attack Status
• Exercise vs Health Risk Analysis
• Interactive Gender Filter
